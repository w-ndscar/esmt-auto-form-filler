# ESMT - Auto Form Filler v1.0.5-beta
This application can be used to automate the process of filling some web forms of the ESMT application with data from an Excel sheet

This app has 3 functions:
- Add Drawings
- Active to Completed
- Convert to Excel

### General Instructions
1. Select the desired function from the dropdown
2. Click **Launch ESMT** and log in to the website
3. Click on **Show Instructions** to display the instructions on how to use

### Add Drawings
***Pre-run check***: When running 'Add Drawings', select the correct project and ensure all elements are present. If any are missing, create them before running the automation — otherwise, the 'Elements' section will appear empty.

1. Follow General Instructions and Login to the ESMT website
2. Select the project and go to the Drawings tab
3. Use **Browse** to select your Excel file
4. Enter the sheet name and specify the starting and ending row numbers
5. Click **Start Automation** to begin

### Active to Completed
1. Follow General Instructions and Login to the ESMT website
2. Select the project and go to the Drawings tab
3. Use **Browse** to select your Excel file
4. Enter the sheet name and specify the starting and ending row numbers
5. Click **Start Automation** to begin
6. For 'Active to Completed', manually click on 'Yes, update it!' button and then press **Esc**

### Convert to Excel (Worktime)
1. Get the CSV from the Worktime app
2. Select **Convert to Excel (Worktime)** from the dropdown
3. Click on Browse, select the CSV File and click Convert
4. The converted Excel will be in the current folder

**Tip**: After filling the values, the app will wait.
Press **Esc** key after each entry to proceed to the next. DO NOT click Submit/New Buttons. It will be done automatically.
